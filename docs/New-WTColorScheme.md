New-WTColorScheme
-----------------
### Synopsis
Creates Windows Terminal Color Schemes

---
### Description

Creates a new Windows Terminal Color Scheme

---
### Related Links
* [Add-WTColorScheme](Add-WTColorScheme.md)



---
### Examples
#### EXAMPLE 1
```PowerShell
New-WTColorScheme -Name MyColorScheme
```

---
